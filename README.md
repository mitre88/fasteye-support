## Common Questions
### Do I need an account?
No. FastEye does not require account creation to use the core experience.

### Does FastEye use cloud AI?
No. Compatible devices can use on-device Foundation Models. If unavailable, the app falls back to local word lists.

### What data is collected?
FastEye is designed to work locally on the device and does not require account-based data collection for the main experience.
